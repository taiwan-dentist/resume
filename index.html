3. 與同事間相處融洽、互相幫忙。甚至可以跟同事互相手術拔掉對方Impaction，再分享討論彼此需要加強的地方，主動自願幫長庚兒牙拍線上Orientation影片......等等。

4. 對治療品質不馬虎。已自購高倍率Univet TTL loupe (4X)、未來會再購入口內攝影相機。

臨床Case紀錄


48 Impaction (深度：Position B + Class II。Root靠在IAN上。)

長庚口腔外科戴醫師：「你明天要幫患者處理這顆？這顆比你自己身上的那個Horizontal impaction還難喔！很深需要移除很多buccal bone。」在充分規劃術式與切線後，隔天順利完成。Check時總醫師表示取出的crown與root切面都很乾淨又平整。 Flap design選擇：Triangular flap。

38 Impaction (Position C + Class I)

Flap design我選擇Envelope到6，因為難度不高。簡單修一點點Buccal Bone後完整頂出智齒，無切牙齒。

48 Impaction (Position A + Class I)

我第一顆Impaction case。上顎18使用拔牙鉗約10秒完成。下顎約1小時(須更多經驗增快速度)，切分牙齒與牙根完成。Flap design：triangular。指導醫師口外戴醫師。

稍微比較彎的上顎彎根智齒

中年男子，無Peri問題。使用細彎形Elevator往Distal傾倒轉出。

Vertucci Type II根管

二合一根管Upper Premolar(發生率22%)，用OD建Wall後完整根管治療，RCF品質經院內主治檢查良好。

我使用電刀做Gingivectomy後在RD下建立滑順完整的Wall，患者同意做Crown所以我Endo完幫Prostho科留了Post space。
