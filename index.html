積極個性、謹慎態度、同事互處、品質堅持

1. 本人在牙科臨床學習上態度積極，在長庚INT期間全程完成zirconia crown、PFM、二合一根管、Impactions(各種不同角度)、也有同一位病人從endo到post到FPD都是我完成的經驗，應該是所有INT裡做到最豐富Case的，會主動跟主治或資深醫師「提早」詢問請教並整理我要做的procedure才主動要Case或施作在患者身上。

2. 在臨床上對待病人謹慎，我所deliver的所有Crown都前一天晚上自己在技工室顯微鏡下修過margin。在長庚牙周病科更是達到離站前的三個禮拜，每個工作天Scaling都有至少一位患者稱讚：「細心」,「仔細」,「沒遇過洗牙那麼用心」,「很溫柔耶。以後可不可以都給你洗？」,「都會先說哪裡會酸又細心教牙線，人很溫柔」等等稱讚。(上過欣美牙科鄭胤捷院長的洗牙與溝通課程)

3. 與同事間相處融洽、互相幫忙。當其他組別只找主治拔Impaction時，甚至可以跟同事互相拔掉對方Impaction再分享討論彼此需要加強的地方，主動自願舉手幫長庚兒牙拍線上Orientation影片......等等。

4. 對治療品質不馬虎。已自購高倍率Univet TTL loupe (4X)、未來會再購入口內攝影相機。

臨床Case紀錄


48 Impaction (深度：Position B + Class II。Root靠在IAN上。)

長庚口腔外科戴醫師：「你明天要幫患者處理這顆？這顆比你自己身上的那個Horizontal impaction還難喔！很深需要移除很多buccal bone。」在充分規劃術式與切線後，隔天順利完成。Check時總醫師表示取出的crown與root切面都很乾淨又平整。 Flap design選擇：Triangular flap。

38 Impaction (Position C + Class I)

Flap design我選擇Envelope到6，因為難度不高。簡單修一點點Buccal Bone後完整頂出智齒，無切牙齒。

48 Impaction (Position A + Class I)

我第一顆Impaction case。上顎18使用拔牙鉗約10秒完成。下顎約1小時(須更多經驗增快速度)，切分牙齒與牙根完成。Flap design：triangular。指導醫師口外戴醫師。
